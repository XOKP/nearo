## Build Setup

``` bash
# install dependencies
$ npm run install

# serve with hot reload at localhost:3000
$ npm run start

# build for production and launch server
$ npm run build
$ npm run start
```

For detailed explanation on how things work, checkout [React.js docs](https://reactjs.org/).
